Supported Data Formats#

Below is a list of all available readers and writers in XSIGMA sorted by extension. Note that for the same extension it could be more than one matching reader/writer since the same extensions are often used across different formats. The list is generated based on a yaml file that contains all the relevant information.

To enable a reader/writer you need to enable the associated module during configuration:

cmake -DXSIGMA_MODULE_ENABLE_<module name>=WANT ...

or setting the flag value via ccmake/cmake-qt.

For example to enable xsigmaPNGWriter which belongs to XSIGMA::IOImage

cmake -DXSIGMA_MODULE_ENABLE_XSIGMA_IOImage=WANT ...

For more details on enabling module see the module system api.

Warning

the list is incomplete, this is work in progress

  • Stanford Exploration Project files reader:

    • Extension: .H

    • reader: xsigmaSEPReader

    • module: XSIGMA::IOImage

  • Alembic scene format:

    • Extension: .abc

    • writer: xsigmaAlembicExporter

    • module: XSIGMA::IOAlembic

  • AVI video files (Windows only):

    • Extension: .avi

    • writer: xsigmaAVIWriter

    • module: XSIGMA::IOMovie

  • LIDAR data using PDAL:

    • Extensions: .bin, .bpf, .csd, .csv, .greyhound, .gpkg, .icebride, .las, .laz, .mat, .nitf, .nsf, .ntf, .pcd, .ply, .pts, .qi, .rxp, .sbet, .sqlite, .sid, .tindex, .txt, .h5

    • reader: xsigmaPDALReader

    • module: XSIGMA::IOPDAL

  • Windows BMP file:

    • Extension: .bmp

    • reader: xsigmaBMPReader

    • writer: xsigmaBMPWriter

    • module: XSIGMA::IOImage

  • FLUENT native format:

    • Extensions: .cas, .dat

    • reader: xsigmaFLUENTReader

    • module: XSIGMA::IOGeometry

  • MotionFX motion definitions cfg files:

    • Extension: .cfg

    • reader: xsigmaMotionFXCFGReader

    • module: XSIGMA::IOMotionFX

  • Computer Graphics Metafile:

    • Extension: .cgm

    • writer: xsigmaCGMWriter

    • module: XSIGMA::IOGeometry

  • CONVERGE CFD CGNS format:

    • Extension: .cgns

    • reader: xsigmaCONVERGECFDCGNSReader

    • module: XSIGMA::IOCGNSReader

  • CGNS format:

    • Extension: .cgns

    • reader: xsigmaCGNSReader

    • module: XSIGMA::IOCGNSReader

  • LS-Dyna databases:

    • Extension: .d3plot

    • reader: xsigmaLSDynaReader

    • module: XSIGMA::IOLSDyna

  • Tabulat data in Tecplot ascii format:

    • Extensions: .dat, .DAT

    • reader: xsigmaTecplotTableReader

    • module: XSIGMA::IOTecplotTable

  • FLUENT CFF format:

    • Extensions: .dat.h5, .cas.h5

    • reader: xsigmaFLUENTCFFReader

    • module: XSIGMA::IOFLUENTCFF

  • DICOM medical images:

    • Extension: .dcm

    • reader: xsigmaDICOMImageReader

    • module: XSIGMA::IOImage

  • Digital Elevation Map File:

    • Extension: .dem

    • reader: xsigmaDEMReader

    • module: XSIGMA::IOImage

  • IOSS (Sierra IO System) files as cell-grids:

    • Extension: .exdg

    • reader: xsigmaIOSSCellGridReader

    • module: XSIGMA::IOIOSS

  • Movie.BYU files:

    • Extension: .g

    • reader: xsigmaBYUReader

    • writer: xsigmaBYUWriter

    • module: XSIGMA::IOGeometry

  • IOSS (Sierra IO System), writer supports only Exodus files:

    • Extension: .g .e .h .gc .ex2 .ex2v2 .exo .gen .par .exoII .exii .ex-timeseries .cgns

    • reader: xsigmaIOSSReader

    • writer: xsigmaIOSSWriter

    • module: XSIGMA::IOIOSS

  • Chaco graph partitioning output files:

    • Extensions: .graph, .coords

    • reader: xsigmaChacoReader

    • module: XSIGMA::IOGeometry

  • VERAout-tools:

    • Extension: .h5

    • reader: xsigmaVeraOutReader

    • module: XSIGMA::IOVeraOut

  • CONVERGE CFD format:

    • Extension: .h5

    • reader: xsigmaCONVERGECFDReader

    • module: XSIGMA::IOCONVERGECFD

  • H5Part particle files:

    • Extension: .h5part

    • reader: xsigmaH5PartReader

    • module: XSIGMA::IOH5Part

  • hdf files generated from xRage, a LANL physics code:

    • Extension: .h5rage

    • reader: xsigmaH5RageReader

    • module: XSIGMA::IOH5Rage

  • GE TRUCHAS format:

    • Extensions: .hdf5, .h5

    • reader: xsigmaTRUCHASReader

    • module: XSIGMA::IOTRUCHAS

  • Radiance HDR file:

    • Extension: .hdr

    • reader: xsigmaHDRReader

    • module: XSIGMA::IOImage

  • AVS UCD Binary/ASCII Files:

    • Extension: .inp

    • reader: xsigmaAVSucdReader

    • module: XSIGMA::IOGeometry

  • JPEG Files:

    • Extensions: .jpg, .jpeg

    • reader: xsigmaJPEGReader

    • writer: xsigmaJPEGWriter

    • module: XSIGMA::IOImage

  • LIDAR data in LAS format:

    • Extension: .las

    • reader: xsigmaLASReader

    • module: XSIGMA::IOLAS

  • binary UNC meta image data:

    • Extensions: .mhd, .mha

    • reader: xsigmaMetaImageReader

    • writer: xsigmaMetaImageWriter

    • module: XSIGMA::IOImage

  • NetCDF-based medical image developed at [BIC:

    • Extension: .mnc

    • reader: xsigmaMINCImageReader

    • writer: xsigmaMINCImageWriter

    • module: XSIGMA::IOMINC

  • H.264-encoded MP4 files (Windows only):

    • Extension: .mp4

    • writer: xsigmaMP4Writer

    • module: XSIGMA::IOMovie

  • MRC Image Files:

    • Extensions: .mrc, .ali, .st, .rec

    • reader: xsigmaMRCReader

    • module: XSIGMA::IOImage

  • NetCDF UGRID file:

    • Extensions: .nc, .ncdf
