## Main Objectives

The ASV volatility fitter provides efficient and robust calibration of volatility surfaces with the following main objectives:

- **Optional repo curve calibration** for American and European option quotes
- **Application of no-arbitrage constraints** through cost and constraint functions
- **Smooth parameter extrapolation** beyond the last liquid maturity
- **Quality control** of the fit and its stability using bid/ask implied volatility data

## Implementation Highlights

- **Uses Brent algorithm** for robust root finding
- **Employs NAG E04UFA routine** for constrained optimization
- **Supports multiple fitting stages** : local, term structure, and global fitting
- **Includes comprehensive error handling** and fallback mechanisms
