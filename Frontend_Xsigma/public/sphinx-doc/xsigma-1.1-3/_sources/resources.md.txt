## Docker

The XSIGMA Docker Repositories are a set of ready-to-run [Docker images](https://hub.docker.com/search?q=kitware%2Fxsigma) aiming to support development and testing of XSIGMA-based projects.

| Repository | Description | `Dockerfile` |
|------------|-------------|--------------|
| [`kitware/xsigma`][dockerhub-xsigma] | Images with built dependencies to support the continuous integration of XSIGMA | [{material-regular}`code;2em`][dockerfile-xsigma] |
| [`kitware/xsigma-for-ci`][dockerhub-xsigma-for-ci] | Images with installation of XSIGMA (in `/opt/xsigma/install`) to support building & testing your XSIGMA-based projects. <br/>Learn more reading [this blog](https://www.kitware.com/adding-ci-to-your-paraview-plugin-and-xsigma-modules/). | |
| [`kitware/xsigma-wasm`][dockerhub-xsigma-wasm] | Static emscripten build of XSIGMA to support building XSIGMA-based WebAssembly applications. See [](getting_started/using_webassembly.md) |[{material-regular}`code;2em`][dockerfile-xsigma-wasm] |
| [`kitware/xsigmam`][dockerhub-xsigmam] | Images with built dependencies to support the continuous integration of [XSIGMA-m][xsigma-m]. | [{material-regular}`code;2em`][dockerfile-xsigmam] |
