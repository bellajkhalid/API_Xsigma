# Using Javascript

[xsigma.js](https://kitware.github.io/xsigma-js/docs/) is an implementation of XSIGMA in JavaScript that consists of an ES6 class library which can be integrated into any web application. See [here](https://kitware.github.io/xsigma-js/docs/#What-is-the-difference-with-XSIGMA-C)  to learn more about the differences between XSIGMA C++ and xsigma.js.
