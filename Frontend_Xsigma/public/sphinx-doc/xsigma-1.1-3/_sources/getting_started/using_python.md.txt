# Using Python

XSIGMA is available on [PyPI](https://pypi.org/) for Windows, macOS and Linux.
```
pip install xsigma
```
or in a [virtual environment](https://docs.python.org/3/library/venv.html) if you want to install the package only locally instead of system-wide

::::{tab-set}

:::{tab-item} Linux

```
python -m venv ./env
source ./env/bin/activate
pip install xsigma
```
:::

:::{tab-item} macOS

```
python -m venv ./env
source ./env/bin/activate
pip install xsigma
```
:::

:::{tab-item} Windows
Using `PowerShell`
```
python -m venv env
.\env\Activate.ps1
pip install xsigma
```

or using `cmd.exe`
```
python -m venv env
.\env\activate.bat
pip install xsigma
```
:::

::::

To verify the installation try to import xsigma from an interactive python environment:

```python
>>> import xsigma
>>> print(xsigma.__version__)
9.2.6
```

That's it ! You may now try some of the
[tutorials](https://kitware.github.io/xsigma-examples/site/Python/#tutorial),
[how to guides](https://kitware.github.io/xsigma-examples/site/PythonHowTo) or
[examples](https://kitware.github.io/xsigma-examples/site/Python).

If you are looking for a higher-level interface to XSIGMA in Python, you may want
to explore using [PyVista](https://docs.pyvista.org) as it exposes XSIGMA in a
"Pythonic" manner.

If you are running the script in a Linux/Windows machine without a display or a GPU, XSIGMA will automatically
select an appropriate OpenGL render window class. Please learn more about how you can influence the
render window selection process in [](/advanced/runtime_settings.md#opengl)
