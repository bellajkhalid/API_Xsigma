Test and Outcome analyses#

Overview#

Cette section présente les résultats des tests de validation du modèle LSV Markov Switching Monte Carlo, incluant les tests de convergence, de précision et de performance.

Test Results Summary#

Convergence Validation#

✅ Convergence confirmée : Les PV du modèle LSV Markov Switching Monte Carlo convergent vers le benchmark dans une plage d’erreur acceptable

✅ Absence de biais : Aucun biais n’a été observé parmi les séries de tests

⚠️ Erreur Monte Carlo : Certains cas sont sujets à une erreur de simulation Monte Carlo plus élevée comparée à un modèle de volatilité locale pure, mais l’erreur est généralement comparable aux autres modèles LSV comme le modèle LSV Hybrid

Performance Benchmarks#

Comparison with Analytical Pricer#

  • Vanilla Options : Convergence vers prix Black-Scholes pour paramètres appropriés

  • Error Range : < 0.1% pour options ATM avec 100K simulations

  • Stability : Résultats stables across différents seeds

Comparison with LSV MS PDE#

  • Consistency : Accord excellent avec pricer PDE pour payoffs supportés

  • Error Range : < 0.05% pour options vanilles européennes

  • Calibration : Utilisation des mêmes paramètres calibrés

Test Coverage#

Product Types Tested#

  • European Calls/Puts : Toutes maturités et strikes

  • Cash-settled Options : Différents délais de règlement

  • Path-dependent Products : Validation sur produits exotiques

Market Scenarios#

  • Normal Markets : Conditions de marché standard

  • Stressed Markets : Volatilité élevée et corrélations extrêmes

  • Low Volatility : Régimes de faible volatilité

  • High Volatility : Régimes de forte volatilité

Parameter Ranges#

  • Spot Levels : 50% à 200% du niveau initial

  • Volatilities : 5% à 50% de volatilité implicite

  • Maturities : 1 semaine à 5 ans

  • Interest Rates : -1% à 10%

Detailed Test Results#

Section 8.6 - Convergence Tests#

Monte Carlo Convergence#

  • Paths Tested : 1K, 5K, 10K, 50K, 100K, 500K

  • Convergence Rate : O(1/√N) conforme à la théorie

  • Standard Error : Diminution appropriée avec augmentation des chemins

  • Stability : Résultats stables pour N ≥ 50K

Greeks Convergence#

  • Delta : Convergence excellente pour tous les produits testés

  • Gamma : Convergence satisfaisante avec N ≥ 100K

  • Vega : Convergence rapide, stable dès N = 50K

  • Higher Order Greeks : Convergence plus lente, N ≥ 200K recommandé

Section 8.7 - Accuracy Analysis#

Pricing Accuracy#

  • ATM Options : Erreur moyenne < 0.05%

  • OTM Options : Erreur moyenne < 0.1%

  • ITM Options : Erreur moyenne < 0.08%

  • Long Maturity : Erreur légèrement plus élevée mais acceptable

Greeks Accuracy#

  • Delta Accuracy : Erreur relative < 1% pour la plupart des cas

  • Gamma Accuracy : Erreur relative < 2% avec paramètres appropriés

  • Vega Accuracy : Erreur relative < 1.5% pour options vanilles

Section 8.8 - Performance Analysis#

Computational Performance#

  • Speed vs LSV Hybrid : 2-3x plus rapide grâce à nature discrète

  • Memory Usage : Efficace, pas de blow-up mémoire observé

  • Scalability : Performance linéaire avec nombre de chemins

  • Parallelization : Excellent scaling sur architectures multi-core

Stability Analysis#

  • Numerical Stability : Aucun blow-up en valorisation observé

  • Calibration Stability : Performance de calibration stable

  • Parameter Sensitivity : Comportement robuste sous variations de paramètres

Validation Against Benchmarks#

Black-Scholes Benchmark#

  • Setup : Paramètres LSV MS configurés pour reproduire BS

  • Results : Convergence vers prix BS avec erreur < 0.01%

  • Greeks : Accord excellent avec Greeks analytiques BS

LSV Hybrid Comparison#

  • Same Calibration : Utilisation des mêmes données de marché

  • Price Comparison : Différences < 0.1% pour la plupart des produits

  • Performance : LSV MS significativement plus rapide

  • Stability : LSV MS plus stable, pas de blow-ups

PDE Benchmark#

  • Consistency Check : Même modèle, différents moteurs de pricing

  • Results : Accord excellent pour payoffs supportés par PDE

  • Error Analysis : Différences attribuables à erreur Monte Carlo uniquement

Risk Management Validation#

Greeks Validation#

  • Finite Differences : Validation croisée avec bumping

  • Pathwise Derivatives : Comparaison avec méthode pathwise

  • Consistency : Greeks cohérents entre méthodes

Scenario Testing#

  • Stress Scenarios : Comportement approprié sous stress

  • Extreme Parameters : Robustesse pour paramètres extrêmes

  • Market Crashes : Gestion correcte des événements extrêmes

Error Analysis#

Monte Carlo Error#

  • Statistical Error : Contrôlé via nombre de simulations

  • Bias Analysis : Aucun biais systématique détecté

  • Confidence Intervals : Intervalles de confiance appropriés

Model Error#

  • Calibration Error : Impact de l’erreur de calibration évalué

  • Parameter Uncertainty : Sensibilité aux paramètres analysée

  • Model Risk : Comparaison avec modèles alternatifs

Numerical Error#

  • Discretization Error : Impact du pas de temps évalué

  • Interpolation Error : Erreur d’interpolation de volatilité locale

  • Rounding Error : Impact négligeable pour précision double

Recommendations#

Production Settings#

  • Number of Paths : 100K pour pricing de production
