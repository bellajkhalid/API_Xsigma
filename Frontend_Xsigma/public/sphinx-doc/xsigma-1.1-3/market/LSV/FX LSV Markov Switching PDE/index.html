FX LSV Markov Switching PDE#

Vue d’Ensemble#

Le modèle FX LSV Markov Switching PDE représente une implémentation avancée du framework LSV Markov Switching utilisant des méthodes d’équations aux dérivées partielles (PDE) pour le pricing précis et efficace d’options FX complexes.

Caractéristiques Principales#

✅ Méthodes PDE Avancées : Différences finies, éléments finis, méthodes spectrales ✅ Spécialisation FX : Optimisé pour marchés de change avec Garman-Kohlhagen étendu ✅ Greeks Directs : Calcul direct des sensibilités via discrétisation PDE ✅ Stabilité Numérique : Schémas stables et convergents pour production ✅ Performance Optimisée : Solveurs parallélisés et grilles adaptatives

Avantages Techniques#

  • Convergence Contrôlée : Erreur de discrétisation maîtrisée

  • Greeks de Qualité : Sensibilités calculées directement sur la grille

  • Traitement des Barrières : Conditions aux limites exactes pour barrières

  • Stabilité Numérique : Schémas implicites stables

  • Réutilisation de Grilles : Efficacité pour pricing multiple

Modèle Mathématique#

Système PDE FX LSV MS#

Le modèle résout le système d’EDPs couplées :

\[\frac{\partial V_i}{\partial t} + \mathcal{L}_i V_i + \sum_{j \neq i} \lambda_{ij}(V_j - V_i) = 0\]

Opérateur Différentiel FX#

L’opérateur pour l’état \(i\) :

\[\mathcal{L}_i = (r_d - r_f)S\frac{\partial}{\partial S} + \frac{1}{2}\sigma_{eff,i}^2 S^2\frac{\partial^2}{\partial S^2} + \kappa(\theta - V)\frac{\partial}{\partial V} + \frac{1}{2}\nu^2 V\frac{\partial^2}{\partial V^2} + \rho\nu\sigma_{eff,i}SV\frac{\partial^2}{\partial S \partial V} - r_d\]

Volatilité Effective par État#

\[\sigma_{eff,i}(S,t,V) = \sigma_{LV}^{FX}(S,t) \cdot \sqrt{V} \cdot \sigma_{\zeta_i}\]

Conditions aux Limites FX#

Conditions spécialisées pour options FX :

  • S → 0 : Comportement call/put approprié

  • S → ∞ : Croissance linéaire asymptotique

  • V → 0 : Convergence vers solution déterministe

  • V → ∞ : Comportement haute volatilité
