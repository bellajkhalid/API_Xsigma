The Model DBOPT Correlation Skew#

Vue d’Ensemble#

Le modèle DBOPT-CorrelationSkew représente une implémentation sophistiquée de copula de corrélation skew pour le pricing de produits 2D. Ce modèle analytique et Monte Carlo à horizon unique est spécialement conçu pour capturer la corrélation skew mise en évidence par les dual digitals à différents strikes.

Caractéristiques Principales#

✅ Copula TERMSKEW : Approche numérique Monte Carlo avec grille 2D de corrélations ✅ Produits 2D : Dual digitals et options contingentes multi-asset ✅ Sticky Strike : Corrélation fonction des niveaux d’underlying ✅ Ajustement Quanto : Support rigoureux des produits multi-devises ✅ Validation Complète : Tests de convergence et stabilité extensifs

Modèle Mathématique#

Copula TERMSKEW#

Le modèle utilise une grille 2D de corrélations fonction des niveaux des sous-jacents (L₁, L₂) avec décomposition de Cholesky pairwise :

\[X_1^{(1)} = \sqrt{\frac{1 + \rho_k}{2}}Z_1 + \sqrt{\frac{1 - \rho_k}{2}}Z_2\]
\[X_2^{(1)} = \sqrt{\frac{1 + \rho_k}{2}}Z_1 - \sqrt{\frac{1 - \rho_k}{2}}Z_2\]

où ρₖ = ρ(ATM₁, ATM₂) est la corrélation interpolée.

Ajustement Quanto#

Pour un produit quanto où un taux R dénommé en devise d est payé en devise f :

\[V_0^f = DF^f(0, T) \cdot E^{Q_f}[S^d(T)]\]

L’ajustement utilise une copula gaussienne standard pour générer la marginale quanto.

Le modèle se calibre aux dual digitals pour déterminer la grille de corrélation skew :

✅ Stress IR : Comportement lisse sous shifts de courbes ✅ Stress FX : Sensibilités cohérentes avec shifts de spot ✅ Stress Volatilité : Stabilité sous stress de vol FX et Asset ✅ Stress Corrélation : Paramètres de corrélation stables

Expert Judgments#

Le modèle utilise le jugement expert du trading desk pour les paramètres de corrélation skew, car les dual digitals ne sont pas suffisamment liquides pour fournir une estimation précise de la corrélation skew. Une corrélation plate est généralement utilisée en l’absence de données de marché.
