References#

Academic Literature#

Foundational Papers#

Black, F., & Scholes, M. (1973). The Pricing of Options and Corporate Liabilities. Journal of Political Economy, 81(3), 637-654. doi:10.1086/260062

  • Foundational article of modern option pricing theory

Dupire, B. (1994, January). Pricing With a Smile. Risk Magazine, 18-20.

  • Introduction of local volatility model and Dupire equation

Derman, E., & Kani, I. (1994, February). Riding on a Smile. Risk magazine, 32-39.

  • Independent development of local volatility framework with implicit tree approach

Gyöngy, I. (1986). Mimicking the One-Dimensional Marginal Distributions of Processes Having an Ito Differential. Probability Theory and Related Fields, 71(4), 501-516. doi:10.1007/BF00699039

  • Fundamental theorem establishing equivalence between stochastic processes and local volatility models

Reference Books#

Glasserman, P. (2003). Monte Carlo Methods in Financial Engineering. New York: Springer Science + Business Media, Inc.

  • Comprehensive reference on Monte Carlo methods in finance

Jaeckel, P. (2002). Monte Carlo Methods in Finance. Chichester: John Wiley & Sons, Ltd.

  • Practical guide to Monte Carlo methods with implementation focus

Karatzas, I., & Shreve, S. (1998). Brownian Motion and Stochastic Calculus (2nd Ed.). New York: Springer-Verlag.

  • Mathematical foundations of stochastic calculus and Tanaka formula

Morton, K. W., & Mayers, D. F. (2005). Numerical Solution of Partial Differential Equations: An Introduction. Cambridge: Cambridge University Press.

  • Numerical methods for solving partial differential equations

Specialized Technical Papers#

Andersen, L. B., Andreasen, J., & Elizer, D. (2002). Static Replication of Barrier Options: Some General Results. The Journal of Computational Finance, 5(4), 1-25.

  • Static replication of barrier options and pricing implications

Joe, S., & Kuo, F. Y. (2003). Remark on Algorithm 659: Implementing Sobol’s Quasirandom Sequence Generator. ACM Transactions on Mathematical Software, 49-57. doi:10.1145/641876.641879

  • Implementation of Sobol quasi-random sequence generators

Matsumoto, M., & Nishimura, T. (1998). Mersenne Twister: A 623-dimensionally Equidistributed Uniform Pseudo-random Number Generator. ACM Transactions on Modeling and Computer Simulation, 3-30. doi:10.1145/272991.272995

  • Mersenne Twister algorithm for pseudo-random number generation

Matsumoto, M., & Nishimura, T. (1999). Dynamic Creation of Pseudorandom Number Generators. In H. Niederreiter, & J. Spanier, Monte Carlo and Quasi-Monte Carlo Methods 1998 (pp. 56-69). Berlin: Springer-Verlag.

  • Dynamic creation of pseudo-random number generators

Internal Documentation#

Derived Models and Extensions#

Austing, P. (2012). CTR Best-of, Worst-of and Dual Digital : Local Correlation PDE. London: Xsigma Capital.

  • Local correlation model for multi-underlying products

Fritz, M. (2012). 3F LobZCB Local Vol : PDE and Monte Carlo. London: Xsigma Capital.

  • Three Factor model with stochastic interest rates

Sadiq, M. (2012). Local Stochastic Volatility Model Documentation. London: Xsigma Capital.

  • LSV Markov Switching model for stochastic volatility

Technical Documentation#

Oliver, N. (2007). Generic PDE Solver - Mathematical Documentation. London: Xsigma Capital.

  • Mathematical documentation of generic QA PDE solver

QAWebFXMCCalculationPP. (n.d.). Retrieved from QA Server Framework Documentation.

  • Documentation of FX Monte Carlo calculation parameters

                    "Brownian Motion and Stochastic Calculus (2nd Ed.)",
                    {"Karatzas, I.", "Shreve, S."},
                    "Springer-Verlag",
                    "1998",
                    "",
                    "",
                    "Fondements mathématiques du calcul stochastique"
                }}
            };
        }
        
        // Génération de citation formatée
        static std::string format_citation(const std::string& ref_id, 
                                         const std::string& style = "apa") {
            auto refs = get_reference_database();
            if (refs.find(ref_id) == refs.end()) {
                return "Référence non trouvée: " + ref_id;
            }
            
            const auto& ref = refs[ref_id];
            
            if (style == "apa") {
                return format_apa_citation(ref);
            } else if (style == "ieee") {
                return format_ieee_citation(ref);
            } else {
                return format_default_citation(ref);
            }
        }
        
        // Génération de bibliographie complète
        static std::string generate_bibliography(const std::string& style = "apa") {
            auto refs = get_reference_database();
            std::ostringstream bibliography;
            
            bibliography << "## Bibliographie\n\n";
            
            for (const auto& [id, ref] : refs) {
                bibliography << format_citation(id, style) << "\n\n";
            }
            
            return bibliography.str();
        }
        
    private:
        static std::string format_apa_citation(const Reference& ref) {
            std::ostringstream citation;
            
            // Auteurs
            for (size_t i = 0; i < ref.authors.size(); ++i) {
                if (i > 0) citation << ", ";
                if (i == ref.authors.size() - 1 && ref.authors.size() > 1) {
                    citation << "& ";
                }
                citation << ref.authors[i];
            }
            
            citation << " (" << ref.year << "). ";
            citation << "*" << ref.title << "*. ";
            
            if (!ref.journal.empty()) {
                citation << ref.journal << ". ";
            }
            
            if (!ref.doi.empty()) {
                citation << "doi:" << ref.doi;
            }
            
            return citation.str();
        }
        
        static std::string format_ieee_citation(const Reference& ref) {
            std::ostringstream citation;
            
            // Format IEEE simplifié
            for (size_t i = 0; i < ref.authors.size(); ++i) {
                if (i > 0) citation << ", ";
                citation << ref.authors[i];
            }
            
            citation << ", \"" << ref.title << "\", ";
            citation << ref.journal << ", " << ref.year << ".";
            
            return citation.str();
        }
        
        static std::string format_default_citation(const Reference& ref) {
            return format_apa_citation(ref);
        }
    };
}
