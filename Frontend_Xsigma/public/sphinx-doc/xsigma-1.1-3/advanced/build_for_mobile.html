Cross-compiling for Mobile devices#

Tip

For complete build instructions see here.

XSIGMA supports mobile devices in its build. These are triggered by a top-level flag which then exposes some settings for a cross-compiled XSIGMA that is controlled from the top-level build.

iOS builds may be enabled by setting the XSIGMA_IOS_BUILD option. The following settings than affect the iOS build:

  • IOS_SIMULATOR_ARCHITECTURES

  • IOS_DEVICE_ARCHITECTURES

  • IOS_DEPLOYMENT_TARGET

  • IOS_EMBED_BITCODE

Android builds may be enabled by setting the XSIGMA_ANDROID_BUILD option. The following settings affect the Android build:

  • ANDROID_NDK

  • ANDROID_NATIVE_API_LEVEL

  • ANDROID_ARCH_ABI
