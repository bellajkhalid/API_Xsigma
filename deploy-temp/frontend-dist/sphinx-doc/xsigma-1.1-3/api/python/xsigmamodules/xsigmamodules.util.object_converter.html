class xsigmamodules.util.object_converter.xsigmaConverter#

Bases: xlwings.conversion.Converter

Convert an object to a reference. The reference can include the ‘caller’ address to ensure uniqueness (not needed if singleton - caller=False).

The object are stored in a global dict by their key and looked up when reading.

When used on a return value, the converter will transform the value into a reference. If the value is a dict, it will return a reference for each item in the dict.

When used in an input, the converter will return the value that was stored in a reference.
